`org.appulsauce.GameRat1` service live here. `gamerat-proto` regenerates
its zbus proxies from these files.

- `org.appulsauce.GameRat1.Profile.xml` — software-abstracted profile
- `org.appulsauce.GameRat1.Rule.xml`   — focus / game-match rule

None of these exist yet; the directory is reserved.
